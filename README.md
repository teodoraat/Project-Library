# Project-Library
OOP
Проектът ми е с цел създаване на приложение за управление на библиотека 
чрез ООП принципи. 

Класове 
- User
  Класът е за отребителите на библиотеката.Неговата основна роля е
  да съхранява е данните, които всички потребители имат.
- Student
  Класът наследява клас User. Чрез методите BorrowBook и ReturnBook,
  потребителят на библиотеката може да заема и връща книги.
- Librarian
  Класът наследява клас User. Librarian представлява библиотекар,
  който може да добавя книги в библиотеката.
- Library
  Класът Library представлява библиотеката,
  която управлява заемането и връщането на книгите.
  Чрез метода  AddBook(string bookName) добавяме книга.
  Методът ShowAvailableBooks() показва наличните книги.
  С BorrowBook(string bookName) студентът заема книга, апслед това се проверява дали е налична.
  С ReturnBook(string bookName) студентът връща книгата.

  В Main добавяме студент и книги. Когато студентът вземе книга програмата извежда останалите книги,
   а когато студентът върне книгата отново излизат всички налични книги.
